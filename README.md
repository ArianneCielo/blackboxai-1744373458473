# Jose Rizal Website

## Project Overview
The Jose Rizal Website is a tribute to the life and legacy of Jose Rizal, a Filipino nationalist and polymath. The website offers information about Rizal's biography, a collection of his poems, and creative works inspired by his writings. It features a user-friendly interface with a modern design, enhanced by Tailwind CSS.

## Installation
To run the Jose Rizal website on your local machine, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd jose-rizal-website
   ```

2. **Open the HTML files**:
   You can directly open `login.html`, `index.html`, `poems.html`, and `students.html` in any web browser.

## Usage
- Upon visiting the site, users will be greeted with a login screen.
- Enter a username and click "Login" to access the homepage.
- The homepage includes:
  - A biography of Jose Rizal.
  - Navigation buttons to explore his poems and student works.
- Users can toggle dark mode for a different theme.

- **Responsive Design**: The website is designed to be mobile-friendly.
